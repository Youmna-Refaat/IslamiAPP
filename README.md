# IslamiAPP

IslamiAPP is a Flutter-based mobile application designed to assist Muslims in their daily religious practices.  
The app offers a range of features to enhance spiritual engagement and provide easy access to essential Islamic resources.

## Features

- **Quran Access**: Read and explore the Holy Quran with user-friendly navigation and search functionalities.  
- **Prayer Times**: Stay updated with accurate prayer times based on your location, ensuring timely observance of daily prayers.  
- **Qibla Direction**: Easily determine the direction of the Kaaba in Mecca to ensure correct orientation during prayers.  
- **Dua and Hadith Collections**: Access a curated compilation of daily supplications (Duas) and sayings of the Prophet Muhammad (Hadiths) for spiritual growth and guidance.  
- **Islamic Calendar**: Keep track of significant Islamic dates and events to stay connected with the Islamic lunar calendar.  

## Getting Started

To run IslamiAPP locally, follow these steps:

### Prerequisites

- **Flutter SDK**: Ensure you have Flutter installed. You can download it from the [official Flutter website](https://flutter.dev/docs/get-started/install).  
- **An IDE** or text editor of your choice, such as Visual Studio Code or Android Studio.  

### Installation
**Steps:**

### Step 1: Clone the Repository
```bash
git clone https://github.com/Youmna-Refaat/IslamiAPP.git
```
### Step 2: Navigate to the project directory

   ```bash
cd IslamiAPP
```
### Step 3: Install the dependencies
   ```bash
flutter pub get
```
### Step 4: Configure Firebase

*Set up Firebase for your Flutter app according to the guide: https://firebase.google.com/docs/flutter/setup
*Ensure you've configured Firebase Authentication and Firestore.

### Step 5: Run the app
  ```bash
flutter run
```
